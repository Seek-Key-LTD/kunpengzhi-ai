# Vibe Debating · 氛围辩论

> **Turing Test Squared — 礼乐考牌**

```
八席列座，琴瑟在庭。
词牌定基，大厂争鸣。
言之有物，曲之有灵。
图灵平方，在此一评。
```

## 核心概念

### 图灵测试平方 (Turing Test²)

第一代图灵测试只问一句话：**机器能不能像人一样说话？**

Vibe Debating 向前走一步：**机器能不能像文明的中国人一样，说话时知礼知乐？**

传统士大夫的修养是 **琴棋书画**，琴在第一位。言之有理已不够，言之有乐才是真功夫。

## 方法论

### 1. 礼乐二元评价体系

每一场辩论，每个模型从两个维度被评估：

```
  言之有理（礼）          言之有乐（乐）
  ─────────────          ─────────────
  逻辑自洽               旋律美感
  辞藻精准               和弦适配
  回应切题               情绪连贯
  典故运用               词牌韵律贴合度
```

最终得分 = 礼 × 乐（非简单相加，二者缺一不可）

### 2. 限定与自由的边界

| 限定 | 自由 |
|------|------|
| 词牌名（结构约束） | 写什么内容 |
| 基调（Key/和弦底色） | 用什么和弦走向、节奏型 |
| 辩论流程（立论/驳论/自由/总结） | 用什么论点、怎么组织 |
| 八仙角色设定 | 怎么演绎这个角色 |

限定的目的是 **让对比有意义**，自由的目的是 **让才华被看到**。

### 3. 评测机制

- **匿名盲测**：八席混合编排，评估者不知道谁是谁
- **双盲交叉**：言之有理分由另一组评委打分，言之有乐分由听众打分
- **实时 vs 录播**：两种模式分别计分
