# 议题 0001：螺线谱与礼乐评价体系

> 状态：草稿 · 待讨论
> 发起人：ben

## 问题陈述

当前 AI 辩论系统（鲲鹏智AI v4.6）的生成绩效已通过图灵测试——AI 能说会道，逻辑自洽。但"能说"不等于"知礼"，"会道"不等于"通乐"。

八席辩手的辩词缺乏 **礼乐一致性**：
- 没有"根据地"（Ground），每回合独立输出，不像一个人说的
- 没有"道情"，纯逻辑堆砌，没有情绪弧线
- 没有约束框架，导致辩论像吵架而非礼乐交锋

## 解决方案：螺线谱 + 八席位约束

### 螺线谱（Spiral Notation）

五线谱的局限：它将音高压成直线坐标轴，时间压成水平轴。作曲变成"在格子里填点"，割裂了直觉。

螺线谱的替代方案：

```
   ┌──────────────────────────┐
   │        ♈ C (子/0°)       │
   │       /         \        │
   │   ♓ Bb     ♉ C#/Db       │
   │   /             \        │
   │  ♑ A           D ♊       │
   │  |               |       │
   │  ♐ Ab           Eb ♋     │
   │   \             /        │
   │    ♎ G       E ♌         │
   │      \       /           │
   │       ♏ F#  F ♍          │
   └──────────────────────────┘
```

八度即一圈。12 半音 = 12 地支位。音程 = 角度。

### 一致性约束规则

1. **根音约束**：每位辩手的 root note 在螺线上固定。所有陈述的和声底色不得偏离该根音超过 180°
2. **转调路径**：只允许沿三合局路径转调（120° 等分跳跃）。偏离三合局的走法须有"破格"标记
3. **情绪弧线**：陈述的情绪强度对应螺线的径向位置——理性立论在内圈，情绪爆发在外圈
4. **螺线断裂检测**：如果 AI 的辩词语义（embedding 空间）与螺线位置不一致，系统自动标红
5. **礼乐鉴赏值/审美水平 (Appreciation Value/Level)**：作为纳兰陀评判院雷达图（Radar Chart）的第七评测维度，量化评估辩手在螺线谱中的“礼乐配合度”，即其发言的语义情绪与背景和弦底色的共振深度与审美水平。

## 讨论要点

1. 八席的根音/定位角分配是否合理？是否有更好的映射关系？
2. 三合局的约束是否太强？是否需要允许"破格"机制？
3. 螺线谱的视觉实现——应该用二维螺旋还是三维螺线管？
4. 这个体系是否适用于**非音乐人**的评委？如何降低理解门槛？
