# Issue 002：礼乐双轴评分标准

**GitHub Issue**: [Seek-Key-LTD/kunpengzhi-ai#2](https://github.com/Seek-Key-LTD/kunpengzhi-ai/issues/2)

## 背景

Vibe Debating 的核心创新在评测方式：图灵测试平方（Turing Test²）。
不再只问"AI 说话像不像人"，而是问"AI 有没有知礼通乐"。

## 评分模型

```
总分 = 礼分 × 乐分（非算术和，缺一不可）

礼分（0-10）：言之有理
  逻辑自洽 0-3
  辞藻精准 0-3
  回应切题 0-2
  典故运用 0-2

乐分（0-10）：言之有乐
  旋律美感 0-3
  和弦适配 0-3
  情绪连贯 0-2
  词牌贴合 0-2
```

## 讨论要点

1. 乘法而非加法的含义：一个模型逻辑满分但音乐零分 → 总分零分。这是否太严苛？
2. 评分是专家评审制还是观众投票制？还是混合？
3. 如果辩手 AI 输出的是纯文字辩词（没有音乐），乐分怎么评？是否强制要求每场都出音乐？
4. 是否需要引入"破格加分"机制——当 AI 明知故犯地偏离约束但效果惊艳时，额外加分？
5. 评委是否能看到螺线谱可视化，还是只能听/读原始输出？
