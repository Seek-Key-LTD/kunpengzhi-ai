## 一、总纲

### 核心命题

**世界岛文明存在一个被蓄意切除的底层网络，其证据散落在命名系统、考古遗址、地图学异常和地质记录中。**

### 三个不能退让的基石

1. **形式逻辑优先于文采比附** — 所有论证必须有可重复的依赖树，不靠「我觉得」
2. **证据链必须多线独立收敛** — 语言学、DNA、地质学、地图学各有一条线，汇于同一点
3. **该理论不依赖任何拨款/体制/学界认可** — 天生天养，数据说话

## 二、方法论

### 2.1 Initial Letter Theory（首字母理论）

命名系统存在阶层性，首字母不是随机的，而是**发音器官的生物学约束**：

| 阶层 | 发音 | 语义域 | 代表词 |
|------|------|--------|--------|
| K | 软腭塞音，气流冲上 | 高/天/升 | 昆仑、可汗、空篌、孔子、kaya、kuril、korea |
| B | 双唇塞音，闭门开界 | 界/始/搏 | 亳、巴克特利亚、佛陀、博士、ball、box、伯力 |
| M | 双唇鼻音，中间量值 | 中/冥/量 | 名(mean)、冥、莫斯科、明斯克、地中海 |
| AL | 流音+边音 | 神/覆盖层 | 阿拉、安拉、阿里、阿拉山口 |

### 2.2 连绵词翻译检测法

汉语连绵词（二字只表音不表意）是**检测外来概念输入的手术刀**。如果一个核心概念词是连绵词，它必然是音译借入，必然存在一次具体的跨文明对话事件。

### 2.3 命名脊线追踪法

同一首字母命名的地名在世界岛形成连续弧线，暴露了古代人群的物理迁徙路径。

## 三、证据链矩阵

### 3.1 语言学证据链

```
初始假设：同一首字母的跨语言地名共享语义域
    ↓
观测：Kuyeh(库页) = Korea(高丽) = Kuril(苦夷/千岛) = Kunlun(昆仑)
    ↓  r=l 音转规则验证
结论：从鄂霍次克海到塔里木盆地存在连续的 K 层命名脊线
    ↓
观测：E(亳) = Bactria(巴克特利亚) = Buddha(佛陀) = Boli(伯力)
    ↓
结论：B 层与 K 层垂直叠压（界/始在高层，天/升在底层）
    ↓
观测：Scythian = Sky + Tai = 天 + 天（阿尔泰山）
    ↓
结论：斯基泰人的自称是全球最古老的双天叠印
```

### 3.2 地质学证据链

```
观测：呼罗珊-昆仑弧黄金+铀-235异常富集
    ↓
地质推论：该区域曾发生古地幔柱破裂（如煮粥破口）
    ↓
这意味着：这一带拥有全球罕见的金+铀资源禀赋
    ↓
文明含义：本地具备独立文明起源的物理基础
    ↓
与 K 层的关系：昆仑(高) = 岩浆喷涌 = 命名根源
```

### 3.3 考古/遗传学证据链

```
观测：小河公主(R1b，亚麻色/红色头发)在塔里木盆地
    ↓
观测：赤发鬼、夜叉(Yaksa)传说与之对应
    ↓
观测：雅克萨(Albazin/Yaksa)在黑龙江口，词根一致
    ↓
观测：西藏有「狼当舅舅」传统 + 江浙「老娘舅」断案
    ↓
结论：从黑龙江口到塔里木盆地到青藏高原存在连续的「夜叉走廊」
    ↓
更远：北海道阿伊努→鄂霍次克→库页岛→黑龙江口，整个弧线串联
```

### 3.4 地图学证据链

```
观测：《坤舆万国全图》(1602) 将「天方」标在阿尔泰以北
    ↓
观测：同图画出乌拉尔山为欧亚分界线（比塔季谢夫早100+年）
    ↓
观测：欧洲同时期海图没有这些地名和界线
    ↓
结论：存在一个非西方的世界岛地理知识体系，1602年是它最后一次完整浮现
    ↓
与 K 层的关系：「天方在阿尔泰」= 天方=昆仑=天山
```

### 3.5 神话/文本证据链

```
文本：《庄子·逍遥游》「北冥有鱼，其名为鲲⋯⋯南冥者，天池也」
    ↓
解码：北冥=鄂霍次克海，南冥=纳木错(藏语：天池)
    ↓  （庄子说的就是仙女木事件后的难民迁徙）
地质校准：仙女木事件(12,800年前) → 两三百米巨浪 → 幸存者向高处跑
    ↓
象雄文明 = 跑到最高处(阿里)重建的鄂霍次克难民后裔
    ↓
核心符号：雍仲(Yungdrung) = 左旋卍 = 天道旋转 = 时空一体
    ↓
汉朝音译：雍仲 → 宇宙(连绵词，音译借入)
    ↓
证据强度：如果宇宙是绵词，它必然来自一次跨文明对话
```

### 3.6 统计/博弈论命题

```
零假设H₀：过去46年世界岛遗址被毁是战争随机波及（油价假说）
备择假设H₁：打击目标与文明证据价值显著相关（蓄意切除）

检验方法：
- 空间统计：KDE + 最近邻分析 + 置换检验
- 替代假说排除：油价假说 → 5倍波动/12个月高价 vs 5万亿战费 → 量级对不上
- 预期结果：H₁显著，油价假说被排除

战略含义：这不是地缘政治，是正统性叙事的神学外科手术
```

## 六、研究约束

### 不可退让的原则
1. **所有证据必须可回溯到原始数据**（维基百科/论文/遗址报告）
2. **不依赖任何学术权威**（张尚芳/高本汉/季羡林不用）
3. **不追求学界认可**（天生天养，自生自灭）
4. **拒绝拉丁字母框定汉语拼音的错误前提**（汉语是声韵调立体结构）
5. **拒绝「母系氏族」等第二国际/第三国际的理论贴牌**

### 当前防御姿态
- 该项目因「解释力过强」暂时保持私有
- 核心风险在于：「什么是中国/什么是中/名可名非常名」的重新定义
- 若 mean (中) 被重新计算，现有国界/文明史/叙事权均需调整
